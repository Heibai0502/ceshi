# LuoKe Auto v1.0.0

首个公开版本，提供可直接运行的前台自动化脚本与 EXE。

## 版本内容

- 深色游戏化控制面板、设置窗口与首次使用说明页
- 启动/停止热键支持 `Ctrl/Alt/Shift/Win + 其他键`
- 动作表情键支持自定义单键录入
- 白天倒计时支持在界面内调整并自动保存
- 设置会写入本地 `LuoKe.ini`，下次启动自动读取
- 提供 `build.ps1`，可自行重新打包 EXE

## 使用提醒

- 建议以管理员身份运行
- 当前为前台执行模式，运行期间请保持游戏窗口可交互
- 建议先用 `1` 分钟倒计时试跑，确认流程正常后再延长

## 附件

- `LuoKeAuto.exe`

## 已知说明

- `LuoKe.ini` 为本地运行配置文件，不包含在源码仓库内
- 如果游戏内修改过动作表情键，请在脚本设置中同步修改
